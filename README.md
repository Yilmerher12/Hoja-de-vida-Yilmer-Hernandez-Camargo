## Propósito del Repositorio

Este repositorio funciona como un **portafolio inicial** para demostrar mis habilidades en el desarrollo **Frontend** (HTML y CSS). Se presentan las actividades de este trimestre las cuales son la CV y nuestra replica de Transmilenio

El objetivo es aplicar las mejores prácticas de **estructura semántica** y **diseño CSS modular** en diferentes escenarios web.


## Estructura de Proyectos

La arquitectura está diseñada para mantener los proyectos totalmente separados y organizados.

* **`README.md`**: El índice principal del repositorio.
* **`CV/`**: Carpeta que contiene mi proyecto de Hoja de Vida profesional.
* **`Transmilenio/`**: Carpeta que contiene la réplica de la interfaz de usuario de la página de Transmilenio.

### Contenido Estándar de las Carpetas

Cada proyecto (`CV/` y `Transmilenio/`) tiene la siguiente estructura interna:

* **`index.html`**: El archivo principal que define la estructura HTML.
* **`styles/`**: Carpeta que contiene la hoja de estilos CSS principal (`style.css`).
* **`img/`**: Carpeta para todos los activos visuales, iconos e imágenes del proyecto.
